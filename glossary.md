# Fintech & SaaS Technical Glossary

| Term | Definition |
| :--- | :--- |
| **AML** | Anti-Money Laundering; regulations to prevent financial crimes. |
| **KYC** | Know Your Customer; the process of verifying a user's identity. |
| **SaaS** | Software as a Service; a software distribution model. |
